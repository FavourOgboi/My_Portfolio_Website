# My_Portfolio_Website

This is a personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS. It showcases projects, skills, and contact information.

## Features

- Responsive design
- Project and skills showcase
- Contact form
- Modern UI with Tailwind CSS

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/FavourOgboi/My_Portfolio_Website.git
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```

## Deployment

This site can be easily deployed to platforms like Netlify or Vercel.
